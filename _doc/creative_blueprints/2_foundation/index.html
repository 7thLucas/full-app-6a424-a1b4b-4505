The Problem

Negotiations Lost in Messaging Apps

  • Price negotiations scattered across WhatsApp, email, and external messaging — no single source of truth
  • Zero legal audit trail — no enforceable record of agreed pricing or order terms
  • Documents — invoices, B/L, packing lists — shared ad hoc with no access control or version authority
  • Manual MOQ enforcement prone to disputes and costly order revision cycles
The Solution

One State-Driven, Auditable Portal

  • Every order follows a strict 9-state machine — UI actions enforced by backend, never by convention
  • Contextual negotiation chat tied by RFQ ID — full business communication audit trail per order
  • Proforma Invoice, B/L, and Packing List in protected Cloudflare R2 — accessed only by matched buyer at the correct state
  • MOQ validated at API level — enforced on every submission, not just suggested in the UI
